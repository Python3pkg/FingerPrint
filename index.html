About Fingerprint

Fingerprint is a software tool developed at UCSD able to analyze computer applications, and to detect their dependencies and requirements. For a given application installed on a system, Fingerprint can automatically save all its dependencies in a file (called Swirl) along with several other information (full path to dependencies, hash of files, etc.). A Swirl file can subsequently be transferred on another system and it can be used to verify if the new system provides all the dependencies required to execute the application. Furthermore Swirl can be used to monitor a system over a period of time, it is indeed possible to detect if one of the dependencies was modified, or altered since its creation.

Instruction

To install from source you can run

 # python setup.py install

This will install FingerPrint in your python. FingerPrint consist of:
 - a bunch of python modules and submodules inside the FingerPrint module
 - a command line called fingerprint

You will need writing priviledge on some system direcotries.

Otherwise you can simply add to your PATH the ./bin directory of this source code,
and you will be done (on bash
export PATH=$PATH:/bin )
